
E-Commerce React App

                                                                    
##  Features

-  Add/remove products from cart
-  Add/remove products from wishlist
-  Product listing using Bootstrap cards
-  Fully responsive UI
-  State management using React Context API
-  Clean and user-friendly interface


  ##  Tech Stack
  
- **React**
- **Bootstrap 5**
- **React Context API**
- **React Query **
- **HTML5 & CSS3**
